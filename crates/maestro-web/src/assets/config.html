Security note

Jira ticket text is passed to AI agents and is an untrusted prompt surface (prompt injection). Use Jira permissions, branch protection, and code review. Maestro also allowlists acli commands to Jira read / assign / transition unless you add [jira] acli_allowed_extra_prefixes in config.toml.

Only the settings below can be changed from this page. Everything else (Jira projects, git, [[agent_steps]], install commands, models, …) must be edited in config.toml on the server, then restart Maestro.
